User Flow 1 Anime Discovery

User Story

As an anime enthusiast, I want to explore a newly trending anime so that I can decide whether to watch it and add it to my watchlist.

Precondition

The user is logged into AniRec.

Postcondition

The user has gathered enough information about an anime, added it to their "Plan to Watch" list, and identified where they can watch it.

Flow

User: Opens up the AniRec website

Website: Displays anime titles, promotional images, and user comments.

User: Finds an anime that he finds interesting and clicks on the anime.

Website: Loads the anime’s page with details like synopsis, score, episode count, and genres.

User: Reads the synopsis, checks user ratings, and scans reviews.

Website: Displays aggregated user ratings, review highlights, and discussion threads.

User: Clicks on the "Characters & Staff" tab to see the cast and creators.

Website: Shows character bios, voice actors, and production staff.

User: Scrolls down to view related anime recommendations.

Website: Suggests similar anime based on user preferences and genre tags.

User: Clicks on the "Manga" tab to check if the anime is an adaptation.

Website: Displays the original manga or light novel with links to its page.

User: Decides to watch and clicks "Add to List," selecting "Plan to Watch."

Website: Updates the user’s anime list and syncs it with their profile.

User: Checks the "Where to Watch" section for streaming availability.

Website: Provides links to legal streaming platforms.

User: Confirms the decision and plans to start watching later.

User Flow 2 Leaving a Review

User Story

As an anime fan, I want to write a review for an anime I just finished so that I can share my opinion with the community and help others decide whether to watch it.

Precondition

The user has completed watching an anime and logged into AniRec.

Postcondition

The user has successfully submitted a review, which is now visible on the anime's page and available for other users to read, rate, and discuss.

Flow

User: Opens the AniRec homepage.

Website: Displays personalized anime recommendations, user activity, and a search bar.

User: Clicks on the search bar and types the anime title.

Website: Shows real-time search suggestions with anime titles, thumbnails, and ratings.

User: Clicks on the correct anime title from the dropdown.

Website: Loads the anime’s page with synopsis, reviews, ratings, and user discussions.

User: Clicks on the “Write a Review” button.

Website: Loads the review submission form with input fields for title, rating, and written review.

User: Writes a detailed review, rates the anime, and previews the submission.

Website: Displays a preview of the review and provides options to edit or submit.

User: Clicks "Submit Review."

Website: Publishes the review on the anime’s page and adds it to the user’s profile.

User: Checks the review on the anime page and shares it on social media.

Website: Provides social sharing options and tracks review engagement (likes, comments).
