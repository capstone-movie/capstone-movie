Conceptual Model

Entities & Attributes

Profile

  • profileId: uuid
  • profileActivationToken: text
  • profileCreateAt: timestamp
  • profileEmail: text
  • profileHash: text
  • profileUsername: text

Anime

  • animeId: uuid
  • animeTitle: text
  • animeTitleAlt: text
  • animeDescription: text
  • animeGenres: text
  • animeAiredStart: date
  • animeAiredEnd: date
  • animeBroadcast: text
  • animeProducer: text
  • animeDuration: decimal
  • animeEpisodes: smallint
  • animeRating: decimal
  • animeStatus: text
  • animeInstagramUrl: text
  • animeOfficialWebsiteUrl: text
  • animeTwitterUrl: text
  • animeType: text

Review

  • reviewId: uuid
  • reviewAnimeId: foreign key to Anime
  • reviewProfileId: foreign key to Profile
  • reviewAnimeRating: smallint
  • reviewBody: text (Content of the post)
  • reviewCreatedAt: timestamp
  • reviewSpoiler: boolean
  • reviewStatus: text ('drafted' or 'posted')

Genre

  • genreId: uuid
  • genreName: text

AnimeGenres

  • animeGenresAnimeId: foreign key to Anime
  • animeGenresGenresId: foreign key to Genre

WatchList

  • watchListAnimeId: foreign key to Anime
  • watchListProfileId: foreign key to Profile
  • watchListFavorite: smallint
  • watchListHidden: smallint
  • watchListLater: smallint

Relations

  • Profile to Review: One Profile can write many Reviews (1 to many)
  • Profile to WatchList: One Profile can have many WatchList entries (1 to many)
  • Anime to Review: One Anime can have many Reviews (1 to many)
  • WatchList to Anime: One WatchList can have many Anime entries (1 to many)
  • Anime to Genre: One Anime can have many Genres through AnimeGenres (many to many)
  • Genre to Anime: One Genre can be associated with many Animes through AnimeGenres (many to many)
